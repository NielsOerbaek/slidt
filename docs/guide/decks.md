# Decks

> **CLI equivalent:** See [cli/deck.md](../cli/deck.md)

## Create a deck

Click **New** on the Decks page. Enter a title. Language defaults to Danish (`da`).

## Duplicate a deck

Click **DUP** on any deck row. A deep copy is created: all slides, the assigned theme, and any deck-scoped templates. No agent history is copied. You'll be redirected to the new deck.

## Apply a theme

In the deck editor top bar, use the Theme dropdown to select a theme. The preview updates immediately. Themes define the color palette and the agent's tone via a system prompt. See [Themes](themes.md).

## Share (read-only link)

Click **Share** in the top bar → **Links** tab → **Create link**. Optionally set an expiry date. Share the URL — anyone with it can view the deck (no login required).

## Add collaborators

Click **Share** → **Collaborators** tab. Enter a colleague's email. They must already have a slidt account. Choose role:
- **Editor** — can view, edit slides, use agent, export
- **Viewer** — can view and export only

Shared decks appear in the collaborator's Decks list under "Shared with me".

## Delete a deck

Click the deck's detail page → **Delete** button in the top bar. This permanently removes all slides, agent history, and share links.

## Export PDF

Click **Export PDF** in the top bar. The PDF is rendered at 1920×1080 per slide.
