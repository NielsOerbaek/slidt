# Slides

> **CLI equivalent:** See [cli/slide.md](../cli/slide.md)

## Adding slides

Click **+** at the bottom of the slide list to pick a slide type. The new slide appears at the end of the deck.

## Editing slide fields

Click a slide in the list to open its fields in the editor panel. Fields vary by type — text, bullets, selects, images. Changes are saved automatically on blur.

**Important:** Never put HTML markup in text fields. The template handles formatting. Plain text only.

## Reordering

Drag slides in the slide list to reorder them.

## Deleting a slide

Click the **x** next to a slide in the list.

## Built-in slide types

See [Slide Types Reference](../reference/slide-types.md) for all built-in types with their field schemas.

## Custom templates

The agent can create deck-scoped slide types (templates) on demand. You can also create and validate them via CLI — see [cli/template.md](../cli/template.md).
