# Getting Started

> **CLI equivalent:** See [cli/README.md](../cli/README.md)

## 1. Log in

Navigate to your slidt instance. Enter your email and password. Contact your admin if you don't have an account — see [Admin](admin.md).

## 2. Create your first deck

Click **New** on the Decks page. Give it a title and press Enter (or the Create button). You'll land on the deck editor.

## 3. Add slides

Click the **+** button at the bottom of the slide list to pick a slide type. See [Slides](slides.md) for the full list of built-in types.

## 4. Use the agent

Press **Cmd+K** to open the agent drawer. Type a prompt like:

> "Create a full pitch deck for a cooperative with a title slide, 3 content slides, and a closing."

The agent will use tools to add and configure slides. Watch the tool calls stream in real time.

## 5. Undo agent actions

The agent drawer shows all undoable actions from the current session. Click **UNDO** next to any action to reverse it.

## 6. Export

Click **Export PDF** in the top bar to download a PDF of your deck.
